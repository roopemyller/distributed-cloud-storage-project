
### Create .venv under the client

`python -m venv .venv`

### Activate venv

`source .venv/bin/activate`(Linux/MacOS)

or

`.venv\Scripts\activate`(Win)

### Install requirements

`pip install -r requirements.txt`

### Install CLI client locally

`pip install -e .`

### Help with cloudcli

`cloudcli --help`
